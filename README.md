🌍 Terrsense
AI-Based Land Quality & Disaster Prediction System

Terrsense is a web-based application that analyzes land conditions and predicts environmental risks such as floods and droughts using historical climate data and basic AI techniques. The goal of this project is to provide a simple and user-friendly platform for understanding environmental patterns and supporting better decision-making.

🚀 Features
🌱 Land quality analysis
🌊 Flood risk prediction
☀️ Drought prediction
📊 Data visualization using charts and graphs
🗺️ Interactive and user-friendly dashboard
📡 Real-world climate data integration
🧠 How It Works

Terrsense follows a simple workflow:

Data Collection
Fetches environmental data (temperature, rainfall, humidity) from NASA POWER API
Data Processing
Cleans and organizes the data for analysis
Prediction
Uses basic AI logic to identify patterns and predict risks
Visualization
Displays results through charts and a dashboard
🛠️ Technologies Used
Frontend: HTML, CSS, JavaScript
Data Source: NASA POWER API
Deployment: Vercel
Tools: VS Code, GitHub
